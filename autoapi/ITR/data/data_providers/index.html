class ITR.data.data_providers.CompanyDataProvider(**kwargs)

Bases: abc.ABC

Company data provider super class. Data container for company specific data. It expects both Fundamental (e.g. Company revenue, marktetcap etc) and emission and target data per company.

Initialized CompanyDataProvider is required when setting up a data warehouse instance.
